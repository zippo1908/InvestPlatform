# PE/VC 投资管理系统功能等价复刻 PRD

## 1. 复刻边界

目标是从零实现一个同类 PE/VC 投资管理系统，达到业务功能等价，而不是复制目标产品的源码、页面布局、图片、Logo、商标、文案、接口、数据库结构或客户数据。

本 PRD 基于两类输入：

- 授权账号下的只读页面观察：菜单、模块、字段、表格列、表单结构、详情页关系。
- 公开资料核对：VC800/PEPM 官方产品介绍与行业竞品资料。

严禁纳入复刻范围：

- 原站前端资源、CSS、图片、图标、Logo、商标、文案。
- 原站接口路径、参数命名、数据库字段名、权限标识。
- 原站客户、项目、基金、联系人、金额、文档、流程记录。
- 原站页面中暴露的密钥、Token、第三方配置。观察过程中发现存在前端敏感配置暴露风险，应作为安全反面案例处理。

## 2. 产品定位

面向私募股权、VC、母基金、政府引导基金、产业基金、投后管理团队的一体化投资管理平台，覆盖“募、投、管、退”和内部协同。

核心价值：

- 以项目和基金为中心，沉淀投资全生命周期信息。
- 将项目、基金、投资人、现金流、估值、流程、文档打通。
- 支持项目看板、审批流程、投后数据收集、风险条款提醒、AI 文档解析。
- 提供管理层驾驶舱和可配置报表。

## 3. 目标用户

- 管理合伙人/领导层：看项目储备、基金表现、投资组合、风险、待办。
- 投资经理：录入项目、推进阶段、上传材料、发起流程、记录会议。
- 基金运营/财务：管理基金基础信息、认缴实缴、现金流、财报、分配。
- 投后经理：收集被投企业数据、维护估值、风险、三会和关键条款。
- 风控/法务：流程审批、协议条款、用印、突发风险、文件归档。
- IR/投资人关系：维护 LP/投资人、出资分配、披露材料。
- 行政/OA：公告、日程、日常流程、文档共享。
- 系统管理员：用户、角色、权限、字段配置、流程配置、审计。
- 外部填报人：被投企业或基金方按邮件/链接填报数据。

## 4. 信息架构

MVP 一级菜单建议：

- 工作台
- 项目管理
- 基金管理
- 投后管理
- 流程中心
- 文档中心
- 风险管理
- 投资人
- AI/投研
- 系统管理

目标产品授权视角下观察到的核心可见模块：

- 首页/驾驶舱、AI 大模型工作台
- 通知公告、日程事件、消息/待办
- 流程中心：项目类、基金类、日常办公类
- 项目池：看板、项目列表、新增、导入、导出
- 项目详情：概况、日程、基金投资情况、权益变动、三会、现金流、估值、投后数据、AI 协议条款、流程文件、文档
- 基金池：基金列表、基金新增、日程、投资人现金流、项目现金流、其他现金流、基金投资组合、基金财报数据、流程文件、文档
- 纪要解析、权益变动、投后数据收集、投资信息
- 投资人、管理机构、流程文件、风险管理、文档管理、回收站、账户设置
- 行研智库、内部产研

## 5. 核心模块需求

### 5.1 登录、账户与安全

功能：

- 账号/邮箱/工号登录。
- 密码重置。
- 30 天自动登录。
- 手机验证码或 MFA，可作为 V1 增强。
- 账户资料、修改密码、偏好设置。
- 登录审计、失败次数限制、设备和 IP 记录。

验收：

- 用户可用账号密码登录并进入工作台。
- 弱密码、过期密码、异常登录有提示和审计。
- 所有会话 Cookie 使用 `HttpOnly`、`Secure`、`SameSite`。
- 前端不得下发任何服务端密钥或第三方 API Secret。

### 5.2 工作台/驾驶舱

功能：

- 待办、已办、我的申请、抄送、未读消息。
- 快捷入口：项目、基金、日程、公告、流程、文档。
- 指标卡：项目阶段分布、基金规模、累计投资、回款、退出、风险提醒。
- 可配置组件和个人常用菜单。

MVP：

- 待办消息、快捷入口、项目/基金概览卡。

V1：

- 管理层驾驶舱、图表筛选、按角色定制首页。

### 5.3 项目池

列表/看板：

- 项目阶段看板：入库、立项、TS、尽调、内审、投决、投资协议、打款、投后服务等。
- 项目列表：搜索、高级筛选、显示列、导入、导出、新增。
- 可按负责人、城市、行业、阶段、入库时间筛选。

项目新增关键字段：

- 商业计划书等文件。
- 企业简称、企业全称、统一社会信用代码。
- 项目状态、项目入库时间、项目负责人。
- 项目所在城市、公司成立日期、行业方向、公司注册地。
- 项目简介、项目亮点、主要产品、其他情况。

项目详情子模块：

- 概况。
- 日程。
- 基金投资情况。
- 权益变动。
- 三会/会议。
- 现金流。
- 估值。
- 投后数据。
- 协议条款 AI。
- 流程文件。
- 文档。

验收：

- 项目可创建、暂存、提交、编辑、删除、恢复。
- 项目阶段可通过看板拖拽或操作变更，变更留审计。
- 项目详情下的日程、现金流、文档、流程均按项目自动过滤。

### 5.4 会议纪要/纪要解析

列表字段：

- 标题、关联项目、会议状态、参会人员、会议时间。

新增字段：

- 标题、关联项目、会议状态、参会人员、会议时间。
- 会议文件 AI 解析。
- 会议议题摘要、决策结果、讨论要点、下一步计划、备注、附件。

功能：

- 上传会议材料。
- AI 提取议题、结论、待办。
- 人工修改后保存。
- 关联项目详情。

MVP：

- 文件上传、纪要表单、关联项目、附件。

V1：

- AI 摘要和行动项提取。

### 5.5 基金池

列表字段：

- 基金简称、基金状态、募集方式、注册资本、基金净资产规模、单位净值、是否托管。
- 投资者人数、主体类型、基金估值、认缴规模、实缴总额、累计投资金额、累计已投企业个数。

基金新增字段域：

- 基础信息：基金简称、基金全称、产品 S 码、基金状态、募集方式、主体类型、信用代码、组织形式、基金类型、基金负责人。
- 规模与出资：注册资本、基金净资产规模、单位净值、目标规模、认缴金额、实缴金额、货币类型、汇率。
- 合规备案：是否托管、是否中基协备案、备案号、是否有引导基金出资。
- 期限：投资期、退出期、延长期、成立日、到期日、LPA 签署时间。
- 策略：投资方式、是否投资基金产品、是否投资单一标的、投资阶段、投资领域、投资限制。
- 管理费：条款类型、计提基数、管理费比例、管理费约定。
- 收益分配：分配原则、分配频率、门槛收益、业绩报酬。
- 治理：投委会成员、表决规则、关键人士、观察员。
- 信息披露：年度报告、季度报告、披露时限。
- 审计和关联交易：审计安排、关联交易风险控制。

基金详情子模块：

- 概况、日程。
- 投资人现金流。
- 项目现金流。
- 其他现金流。
- 基金投资组合。
- 基金财报数据。
- 流程文件。
- 文档。

验收：

- 基金可创建、保存草稿、提交、编辑。
- 金额、比例、日期字段有格式校验。
- 基金下现金流、投资组合、财报、文档按基金自动过滤。

### 5.6 投资信息/投资组合

列表字段：

- 项目名称、投资主体、协议签署金额、首次投资日期。
- 累计打款金额、剩余投资成本。
- 初始持股比例、最新持股比例、最新持有估值。
- 累计回款金额、累计退出成本、累计退出收益、累计退出分红。
- 首次回款时间、最新回款时间、完全退出时间、退出方式、退出状态。

功能：

- 记录基金对项目的投资关系。
- 支持协议金额、实际打款、持股比例、估值、回款、退出状态。
- 自动汇总至项目详情和基金详情。

### 5.7 权益变动

列表字段：

- 项目名称、投资主体、股权变更原因、协议时间、投决会通过时间。
- 轮次、是否领投、投资方式、交易前股比、交易后占比、本轮其他投资机构。

功能：

- 记录融资轮次和股权比例变化。
- 与投资信息、估值、项目阶段联动。

### 5.8 投后数据收集

列表字段：

- 项目名称、收件邮箱、收集状态。

功能：

- 批量发送填报邮件。
- 重置填报状态。
- 外部填报链接。
- 填报数据回流至投后数据表和项目详情。

MVP：

- 内部手工录入和状态跟踪。

V1：

- 邮件发送、外部填报、自动催收。

### 5.9 风险管理

关键条款字段：

- 基金简称、项目名称、轮次、条款状态、特殊条款、条款内容、提醒日期。

功能：

- 项目关键条款台账。
- 提醒日期触发待办和消息。
- 突发风险事务登记。
- 关联协议、项目、基金和责任人。

### 5.10 流程中心

分类：

- 项目类流程。
- 基金类流程。
- 日常办公类流程：行政类、人事类、其他类。

功能：

- 流程发起、审批、抄送、转办、委托设置。
- 流程与项目/基金/文件关联。
- 审批记录、附件、意见、水印、导出。

MVP：

- 固定流程模板：项目立项、投决、用印、付款、合同审批。

V1：

- 可视化流程引擎、表单配置、条件分支、委托审批。

### 5.11 文档中心

分类：

- 项目类文件。
- 基金类文件。
- 共享类文件。
- 流程文件。

功能：

- 文件夹、上传、预览、下载、版本、权限、水印、回收站。
- 文档关联项目、基金、流程、会议、风险条款。
- 全文检索和标签。

### 5.12 投资人和管理机构

功能：

- 投资人/LP 档案。
- 认缴、实缴、分配、退款、联系记录、披露材料。
- 管理机构/GP 档案。
- 与基金、现金流、文档关联。

### 5.13 AI/投研

功能：

- AI 工作台。
- 行研智库：摘录、标签、文档。
- 内部产研：报告、文档沉淀。
- 会议纪要解析。
- 商业计划书/基金文件解析，自动填充部分字段。
- 协议关键条款提取和风险提醒。

MVP：

- 文档上传、人工摘要字段。

V1：

- AI 提取结构化字段，人工确认后入库。

## 6. 通用列表能力

所有数据模块应具备：

- 关键字搜索。
- 高级搜索。
- 显示列配置。
- 分页、排序。
- 新增、编辑、删除、查看。
- 导入、导出。
- 批量操作。
- 操作审计。
- 按角色、组织、负责人、数据归属控制可见范围。

## 7. MVP 范围

建议 12-16 周做出可用 MVP：

- 登录、用户、角色、组织、审计。
- 工作台：待办、快捷入口、基础指标。
- 项目池：列表、看板、新增、详情、文档、日程。
- 基金池：列表、新增、详情、投资组合。
- 投资信息、权益变动。
- 会议纪要。
- 固定流程模板。
- 文件中心。
- 基础导入导出。
- 风险关键条款提醒。

暂不放入 MVP：

- 完整可视化流程设计器。
- 外部填报门户。
- 完整 AI 自动填表。
- LP 门户。
- 全量驾驶舱和复杂收益测算。
- 移动端完整适配。

## 8. V1 增强范围

- 投后数据外部填报和邮件催收。
- 基金现金流、投资人现金流、其他现金流。
- 估值、财报数据、退出收益。
- 自定义字段/视图/表单。
- 自定义流程引擎。
- AI 文件解析、会议纪要、协议条款提取。
- 报表驾驶舱。
- 文档全文检索、水印和权限增强。

## 9. V2 企业版范围

- 多租户/多空间。
- 移动端。
- LP 门户/被投企业门户。
- SSO、企业微信/飞书/钉钉/Outlook 集成。
- 私有化部署包。
- API 开放平台。
- 数据仓库和 BI。
- 高级收益测算：IRR、DPI、TVPI、MOIC、Carry、J-Curve。

## 10. 非功能需求

- 权限：RBAC + 数据范围 + 字段级权限 + 文档权限。
- 审计：所有增删改查、导出、下载、审批、登录都留痕。
- 安全：密钥只在服务端，前端不得下发 Secret；文件防越权；下载水印。
- 合规：支持私有化、备份恢复、日志留存、数据脱敏。
- 性能：常用列表 1 秒内首屏返回；大导入异步处理。
- 可维护性：模块配置化，但核心投资对象保持强模型。
