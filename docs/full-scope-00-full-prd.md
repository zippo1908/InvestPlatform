# PE/VC 投资管理系统全量 PRD

## 1. 目标

从零实现一个功能等价的 PE/VC 投资管理系统，覆盖募、投、管、退和内部协同的完整业务闭环。该系统不复制目标站源码、素材、商标、页面文案、接口路径、数据库结构、客户数据或敏感配置。

## 2. 全量模块

1. 入口 / 登录页：账号、密码、验证码、自动登录和安全提示
2. 工作台 / 管理层工作台：待办、关键指标、项目阶段、基金组合和风险热区
3. 工作台 / AI 大模型工作台：文件解析、会议摘要、协议条款提取和投研问答
4. 协同工具 / 通知公告：公告列表、发布、导入、导出和阅读范围
5. 协同工具 / 日程事件：周日程、月日程、列表和关联项目/基金日程
6. 协同工具 / 消息中心：未读、全部、我的待办、我的已办、我的申请和抄送
7. 流程中心 / 流程中心总览：项目类、基金类、日常办公类流程入口和委托设置
8. 流程中心 / 项目类流程：立项、尽调、投决、投资协议、打款、退出流程
9. 流程中心 / 基金类流程：基金设立、付款、LP 披露、条款变更和财报审批
10. 流程中心 / 日常办公流程：行政、人事、用印、报销和请假流程
11. 项目管理 / 项目池看板：入库、立项、TS、尽调、内审、投决、协议、打款、投后服务
12. 项目管理 / 项目列表：高级搜索、显示列、导入、导出、新增和批量操作
13. 项目管理 / 新增项目：BP 上传、AI 解析填充、项目基础信息和描述字段
14. 项目管理 / 项目详情-概况：项目主档、阶段历史、负责人、摘要和关键动作
15. 项目管理 / 项目详情-投资关系：基金投资情况、权益变动和估值记录
16. 项目管理 / 项目详情-投后数据：月度/季度经营数据、财务指标、填报状态和附件
17. 项目管理 / 纪要解析：会议文件上传、AI 摘要、决策结果、讨论要点和下一步计划
18. 基金管理 / 基金列表：基金状态、规模、净值、托管、认缴实缴和累计投资
19. 基金管理 / 新增基金：基础信息、规模、期限、投资策略、管理费、分配、治理和披露
20. 基金管理 / 基金详情-概况：基金条款、规模、出资节奏、组合表现和关键时间
21. 基金管理 / 基金详情-现金流：投资人现金流、项目现金流、其他现金流和回款分配
22. 基金管理 / 基金详情-财报数据：基金财务报表、净值、估值和披露材料
23. 基金管理 / 投资信息：项目投资台账、协议金额、打款、持股、回款和退出状态
24. 基金管理 / 权益变动：股权变更原因、协议时间、轮次、领投、投资方式和股比变化
25. 投资人 / 投资人列表：LP/投资人档案、认缴实缴、联系人和披露状态
26. 投资人 / 投资人详情：档案、基金份额、现金流、联系记录和披露文件
27. 投资人 / 管理机构：GP、管理人、执行事务合伙人和关联基金
28. 投后管理 / 投后数据收集：外部填报、邮件发送、填报状态重置和数据回流
29. 风险管理 / 项目关键条款：基金、项目、轮次、特殊条款、条款内容、提醒日期和状态
30. 风险管理 / 突发风险事务：风险事件登记、等级、处置方案、负责人和进展
31. 文档管理 / 文档中心：项目类文件、基金类文件、共享类文件、流程文件和权限水印
32. 文档管理 / 流程文件：从流程归档的审批文件、合同、用印材料和付款附件
33. AI数据库 / 行研智库：行研摘录、标签、来源、AI 摘要和关联项目
34. AI数据库 / 内部产研：内部报告、研究文档、附件、评审和知识沉淀
35. 报表驾驶舱 / 报表驾驶舱：IRR、DPI、TVPI、MOIC、行业分布、阶段分布和资金流
36. 通用能力 / 导入导出中心：模板下载、导入校验、错误回写、异步导出和任务历史
37. 系统管理 / 用户与组织：组织树、用户账号、状态、角色和登录审计
38. 系统管理 / 角色与权限：菜单权限、操作权限、数据范围、字段级权限和文档权限
39. 系统管理 / 字段与表单配置：自定义字段、显示列、表单布局、选项集和校验规则
40. 账户 / 账户与偏好设置：个人资料、密码、通知偏好、常用菜单和安全设备
41. 系统管理 / 回收站：文件、对象、恢复、彻底删除和操作审计

## 3. 用户角色

- 管理合伙人：查看驾驶舱、项目储备、基金表现、投后风险、审批事项。
- 投资经理：维护项目池、推进阶段、上传材料、发起流程、记录会议。
- 基金运营：维护基金主档、出资、现金流、财报、披露材料。
- 财务：处理付款、管理费、回款、分配、报表。
- 投后经理：收集经营数据、估值、三会、风险条款。
- 风控法务：尽调、协议、关键条款、用印、审批、风险事件。
- IR/投资人关系：维护投资人、沟通记录、披露文件、LP 现金流。
- 研究员：维护行研智库、内部产研、AI 摘要和投研知识。
- 系统管理员：用户、组织、角色、权限、字段、表单、流程模板和审计。

## 4. 全量功能要求

### 4.1 入口与账户

- 账号/邮箱/工号登录、密码登录、验证码登录、30 天自动登录、忘记密码。
- 账户资料、修改密码、通知偏好、设备管理、登录审计。
- 会话安全、弱密码提醒、异常登录提醒。

### 4.2 工作台和消息

- 管理层工作台：项目阶段、基金规模、累计投资、风险、待办。
- 消息中心：未读、全部、我的待办、我的已办、我的申请、抄送。
- 通知公告：发布、阅读范围、导入、导出、阅读状态。
- 日程事件：周视图、月视图、列表、项目/基金关联日程。

### 4.3 流程中心

- 项目类流程：项目立项、尽调、投决、协议、打款、退出。
- 基金类流程：基金设立、付款、LP 披露、条款变更、财报审批。
- 日常办公流程：行政、人事、用印、报销、请假。
- 流程能力：发起、审批、驳回、转办、抄送、委托、附件、归档、审计。

### 4.4 项目管理

- 项目池看板：入库、立项、TS、尽调、内审、投决、投资协议、打款、投后服务。
- 项目列表：高级搜索、显示列、导入、导出、新增、批量操作。
- 新增项目：BP 上传、AI 解析填充、企业简称、全称、信用代码、状态、负责人、城市、行业、注册地、简介、亮点、产品、其他情况。
- 项目详情：概况、日程、基金投资情况、权益变动、三会、现金流、估值、投后数据、协议条款 AI、流程文件、文档。

### 4.5 基金管理

- 基金列表：状态、募集方式、注册资本、净资产、单位净值、托管、投资者人数、认缴、实缴、累计投资、已投企业数。
- 新增基金：基础信息、组织备案、规模期限、投资策略、管理费、收益分配、投委会、关键人士、观察员、信息披露、审计和关联交易。
- 基金详情：概况、日程、投资人现金流、项目现金流、其他现金流、投资组合、基金财报、流程文件、文档。
- 投资信息：协议金额、首次投资、累计打款、持股比例、估值、回款、退出状态。
- 权益变动：变更原因、协议时间、投决时间、轮次、是否领投、投资方式、交易前后股比。

### 4.6 投后管理

- 投后数据收集：外部填报、邮件发送、状态重置、催收、回流。
- 项目经营数据：营收、净利润、现金余额、员工数、估值、附件。
- 三会和会议纪要：上传文件、AI 解析、人工确认、关联项目。

### 4.7 投资人和管理机构

- 投资人列表与详情：LP 档案、基金份额、现金流、联系人、披露材料、沟通记录、文档。
- 管理机构：GP、基金管理人、执行事务合伙人、顾问机构、关联基金。

### 4.8 文档管理

- 项目类文件、基金类文件、共享类文件、流程文件、回收站。
- 文件上传、预览、下载、版本、权限、水印、审计、全文检索。
- 文件可关联项目、基金、投资人、流程、会议、风险条款。

### 4.9 风险管理

- 项目关键条款：基金、项目、轮次、条款状态、特殊条款、内容、提醒日期、责任人。
- 突发风险事务：事件、等级、发生时间、负责人、处置状态、处置方案、附件。
- 风险提醒进入待办和驾驶舱。

### 4.10 AI 与投研

- AI 大模型工作台：会议、BP、合同、报告解析。
- 行研智库：摘录、标签、来源、AI 摘要、关联项目。
- 内部产研：报告、版本、评审、文档沉淀。
- AI 输出必须人工确认后入库。

### 4.11 报表驾驶舱

- 指标：IRR、DPI、TVPI、MOIC、投资阶段分布、行业分布、资金流、风险热区。
- 可按基金、项目、行业、负责人、时间区间过滤。

### 4.12 系统管理

- 用户组织、角色权限、字段和表单配置、导入导出中心、回收站。
- 权限包含菜单、操作、数据范围、字段级、文档级。
- 所有敏感操作审计。

## 5. 验收标准

- PRD 中每个页面在设计文件中都有对应屏幕。
- 页面覆盖列表、新增、详情、看板、流程、文档、配置、导入导出和状态管理。
- 全部设计使用虚构数据和新品牌。
- 不含密钥、Token、真实客户数据、原站素材或原站接口。
